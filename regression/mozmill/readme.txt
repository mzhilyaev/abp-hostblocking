Download mozmill 
$ https://mozqa.com/mozmill-env/

Install mozmill environment
$ mkdir mozmill && cd mozmill && hg clone http://hg.mozilla.org/qa/mozmill-tests && unzip <mozmill.zip> && cd mozmill-env && source ./bin/activate

Run the test
$ ./run.sh
